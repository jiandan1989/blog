```vim
# 配置用户名: git config --global user.name "xxx"

# 配置邮箱: git config --global user.email "xxx@xxx.com"

# git status 等命令行自动着色 :

git config --global color.ui true

git config --global color.status auto

git config --global color.diff auto

git config --global color.branch auto

git config --global color.interactive auto

# remove proxy configuration on git: git config --global --unset http.proxy

# 查看配置: git config --list (也可以通过 git config <key>的方式查看某一项配置)


```


### 常用命令

```vim
git init ; 初始化一个git仓库

git clone : 从远程下载一个git 项目

git branch : 查看本地分支情况

git status : 查看当前文件状态

git status -s, git status --short: 简短状态查看

git add <filename> || git add .: 跟踪新文件 || 跟踪所有修改过的文件

.gitignore: 在根目录下创建一个.gitignore文件,将不需要跟踪或者提交的文件名写进去,忽略该文件

git diff: 查看已暂存和为暂存的修改,比对文件

git diff --cached, git diff --staged: 查看已暂存的将要添加到下次提交里的内容

git commit -m "提交信息": 提交更新

git mv file_from file_to: 移动文件,直接修改文件名,相当于三条命令 git rm file_from, git add file_to

git log : 查看提交记录,可以添加参数, -p 显示每次提交的内容差异,也可以加上 -2显示最近两次提交

git reset HEAD filename: 取消暂存


```
